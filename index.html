从零开始搭建 Hexo + NexT 个人博客（2025 最新）

本系列将带你打造一款兼具“V 字面具”与“小丑”元素的个人博客，分为四大章节：

第一章 博客搭建

  1. 环境准备：安装 Node.js、Git 以及 VS Code。
  2. 安装 Hexo：npm install -g hexo-cli 并初始化项目。
  3. 添加 NexT 主题：克隆 hexo-theme-next，复制配置文件到 _config.next.yml。
  4. 启动预览：hexo s 查看本地效果。

第二章 博客配置

  1. 站点信息：在 _config.yml 中设置标题、描述、关键词等。
  2. 主题菜单：在 _config.next.yml 配置导航，并新增 contact、subscribe 等页面链接。
  3. 博客分类：创建 混沌理论、戏剧沉思、蒙面真相 三大分类，用于技术、哲学及时事文章。
  4. 社交媒体图标：在 social: 部分放置 GitHub 等链接，可自定义 V 形面具或小丑帽图标。
  5. 评论区：启用支持匿名的评论插件，开启审核以维持基调。
  6. 搜索与 404 页面：启用本地搜索，占位符为“搜索混沌…”，并在 source/404.md 定制错误页面。

第三章 配置进阶

  1. 联系页面：/contact/ 页面提供表单，邮箱地址不公开，并加入谜语验证码。
  2. 订阅功能：/subscribe/ 页面让访客输入邮箱，按钮文案为“加入假面舞会”。
  3. 自定义页脚：在 source/_data/footer.njk 添加语录“微笑是混乱的最佳面具”。
  4. 匿名评论：通过 Valine 或 Waline 设置，鼓励读者留言。

第四章 博客优化细节

  1. SEO 与站点地图：安装 hexo-generator-sitemap 等插件并配置。
  2. 性能优化：开启压缩、使用 CDN、图片懒加载等。
  3. 备份与部署：使用 Git 管理源码，hexo g && hexo d 部署到 GitHub Pages。

至此，你的 Hexo + NexT 博客就大功告成，欢迎在评论区留下足迹。
